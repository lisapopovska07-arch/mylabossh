Лабораторна робота №5

Тема
Основи роботи з Git, SSH-ключі, гілки, коміти.

Опис
Мета лабораторної роботи — навчитися налаштовувати віддалений репозиторій, створювати гілки, коміти та керувати файлами.

Мета роботи
Ознайомитися з роботою Git, створенням SSH-ключів, гілками та базовим документуванням проєкту.

Контрольні запитання

1. Що таке Git?
   Розподілена система контролю версій.

2. Для чого потрібен SSH-ключ? 
   Безпечна аутентифікація без пароля.

3. Що таке гілка (branch)?
   Паралельна лінія розробки для внесення змін.
